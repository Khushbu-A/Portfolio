Website
design

InstaList allows the user a more efficient way to organize their grocery list. The user is prompted to a login page where they can login if they have an existing account or register if they do not have an account. When the users register, they are able to select a list of dietary restrictions they might have. Once they are logged in, on the homepage they can search for recipes they might be interested in. The user can then select a recipe and add it to their recipe list, if the user chooses, they can add the ingredients to their grocery list which will be put into their respective categories such as frozen, bakery and more. The user can manually update their grocery list and add in the amount they want for each item. There is also a help tab for users that are unfamiliar with the website and a guide if needed. Users can access their dietary restrictions through the profile page which also includes a link to their recipe and an option to edit their profile. AI Model
Training

This project involved developing a computer vision system designed to help robots identify and interact with objects in their environment. By using machine learning techniques, specifically the YOLO (You Only Look Once) object detection framework, the system can recognize and differentiate between various objects. The process began with creating a custom image dataset and carefully annotating images to teach the model what specific objects look like. The annotated dataset was then used to train the YOLO model, enabling it to accurately detect objects based on visual input. After training, the model was deployed onto a Raspberry Pi—a compact and efficient computing device—allowing for real-time object detection using a connected camera. The system was optimized to run efficiently even on this low-power hardware, making it suitable for real-world applications such as robotics, automation, and interactive systems. Through this project, key skills such as dataset preparation, image annotation, machine learning model training, performance evaluation, and embedded system deployment were developed and applied, demonstrating practical experience in integrating AI with robotics. Data
Analyst

This project explored mixed martial arts (MMA) through data-driven analysis, focusing on identifying factors that contribute to fighter performance and injury prevention. By examining extensive historical fight records, we analyzed various factors like fighter age, fighting stance, striking accuracy, knockdowns, and physical attributes such as height, weight, and reach. Using Python-based machine learning models (including logistic regression, decision trees, and random forests) and visualization techniques, we uncovered patterns that can help coaches and athletes optimize training strategies and enhance fight safety. Our comprehensive approach included careful data preprocessing, statistical analysis, and visualization, providing practical insights for improving strategies in the ring and minimizing injuries.
